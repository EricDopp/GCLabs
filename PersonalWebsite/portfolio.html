Library Terminal Project

A console application that will allow you to browse, checkout, return, and sort books.

Dice Roller App

A console application that will allow you to choose how many dice you wish to throw and how many sides you wish the dice have. There is also built-in functionality for the grame of craps.

Movie Database

A console application that will allow you to browse and sort a list of movies by genre and title.

Circle Creation App

A console application that allows you to input a value for the radius that will then calculate the diamter, area, and circumference of the resulting circle that is created. The circle can then be expanded with the respective values changing appropriately.

Coin Flipping Challenge!

A console application that challenges you to flip a coin and guess whether it's heads, or tails. You can refuse to continue at any time but you are called a coward if you do so at the beginning.
